This project is a template set up with [Next.js](https://nextjs.org/) and [Radix UI](https://www.radix-ui.com/)

> Quick start template for saving time

## Install Library

- [Next.js](https://nextjs.org/)
- [Radix UI](https://www.radix-ui.com/)
- [next-theme](https://github.com/pacocoursey/next-themes#readme)

## Getting Started

- First, install the node_modules

```bash
npm run install
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
- Secon, run the development server

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

- Third, open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Starter Pages

- page `src/app/page.tsx`

- components `src/components/Welcome.tsx`
